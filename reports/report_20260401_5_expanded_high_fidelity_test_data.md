# Android TV 테스트용 프로필 데이터 셋 확장 보고서

## 1. 개요
프로필 선택 화면의 시각적 다양성을 높이고, 보유하신 모든 아바타 리소스(avatar_1 ~ avatar_7)를 테스트에 활용할 수 있도록 테스트 데이터 생성 로직을 확장하였습니다.

## 2. 주요 업데이트 내용
### 2.1 7인 유니크 프로필 데이터 매핑
- **리소스 전수 활용**: 기존 3가지 아바타 반복 대신, `avatar_1`부터 `avatar_7`까지 7개의 리소스를 인덱스별로 1:1 매핑하였습니다.
- **닉네임 다변화**: 각 아바타 캐릭터에 어울리는 한국인 닉네임을 할당하였습니다.
  - 리스트: **지은, 민준, 하나, 서윤, 도윤, 하윤, 시우**
- **전용 컬러 팔레트**: 각 캐릭터의 개성을 살릴 수 있는 7가지 파스텔톤 그라데이션 팔레트(Pink, Blue, Green, Yellow, Purple, Orange, Cyan)를 정의하여 적용하였습니다.

### 2.2 테스트 가동 플로우
- 리모컨 숫자 키 **1, 2, 3, 4, 5**를 눌러 프로필을 확장할 때, 지정된 7명이 먼저 순서대로 표시되며 8번째부터는 순환 참조 또는 익명 사용자로 표시됩니다.
- 이를 통해 다양한 아이콘과 배경색 조합이 아코디언 레이아웃에서 어떻게 어우러지는지 더 실감나게 검증할 수 있습니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 숫자 키 4(7명) 또는 5(10명)를 눌렀을 때, 7가지의 서로 다른 캐릭터 리소소와 닉네임이 정상적으로 표시되는지 확인하십시오.
