# Android TV 텍스트 정보 우측 배치 및 겹침 방지 최적화 보고서

## 1. 개요
화면 중앙에 정렬된 프로필 카드가 확장되거나 이동할 때 배경 문구와 겹치는 문제를 해결하기 위해, 모든 텍스트 기반 정보(타이틀, 가이드)를 화면 우측 사이드로 이동 배치하였습니다.

## 2. 주요 수정 사항
### 2.1 텍스트 위치 우측단 이동 (Right-side Alignment)
- **상단 타이틀**: `Alignment.TopEnd`로 배치하고 `end = 80.dp` 패딩을 적용하여 화면 우측 상단에 고정하였습니다.
- **하단 가이드**: `Alignment.BottomEnd`로 배치하고 `end = 80.dp` 패딩을 적용하여 화면 우측 하단에 고정하였습니다.

### 2.2 우측 정렬 텍스트 디자인 (Text Alignment.End)
- 단순히 위치만 옮긴 것이 아니라, `Column` 내의 `horizontalAlignment`를 `Alignment.End`로 설정하고 각 `Text` 컴포저블의 `textAlign`도 `TextAlign.End`로 맞추어 우측 벽면을 기준으로 세련되게 정렬되도록 하였습니다.

### 2.3 시각적 공간 분리 효과
- **중앙 영역**: 프로필 카드 탐색 및 아코디언 확장을 위한 전용 공간으로 확보되었습니다.
- **우측 영역**: 서비스 타이틀 및 조작 가이드를 위한 정보 제공 공간으로 분리되었습니다.
- **결과**: 어떠한 경우에도 포커스된 프로필 카드와 배경 문구가 겹치지 않아 시각적 완성도가 극대화되었습니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 화면 우측 상단에 "Choose Your Experience" 문구가, 하단에 조작 가이드가 깔끔하게 우측 정렬되어 표시되는지 확인하십시오.
