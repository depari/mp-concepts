# Android TV 프로필 선택 화면 레이아웃 계층(Z-Index) 최적화 보고서

## 1. 개요
프로필 카드가 확장될 때 상단 배경 문구("Choose Your Experience")를 가려 완성도가 떨어지는 문제를 해결하기 위해 컴포저블의 렌더링 계층 구조를 전면 재조정하였습니다.

## 2. 주요 수정 사항
### 2.1 Z-Index 레이어 순서 변경 (Layer Reordering)
- **문제 원인**: 기존에는 문구가 먼저 선언되어 있어, 나중에 선언된 프로필 카드 리스트가 문구 위를 덮는 계층 구조였습니다.
- **해결 방안**: `ProfileSelectionScreen` 의 `Box` 내부 선언 순서를 아래와 같이 변경하여 **문구가 항상 최전방**에 위치하도록 하였습니다.
  1. `TvLazyRow` (프로필 목록 - 배경 레이어)
  2. `Column` (상단 타이틀 - 전면 레이어)
  3. `Column` (하단 가이드 - 전면 레이어)

### 2.2 기대 효과
- **가독성 향상**: 프로필 카드가 아코디언처럼 최대 450dp까지 확장되더라도, 상단 문구가 카드 위로 선명하게 표시되어 사용자가 현재 어떤 화면에 있는지 명확히 인지할 수 있습니다.
- **UI 완성도**: 레이아웃이 겹칠 때 발생하는 불필요한 클리핑(Clipping) 현상을 제거하여 프리미엄 블랙 배경에 어울리는 정교한 디자인을 완성했습니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 앱 시작 시 및 프로필 이동 시 상단 문구가 프로필 카드 전면에 안정적으로 표시되는지 확인하십시오.
