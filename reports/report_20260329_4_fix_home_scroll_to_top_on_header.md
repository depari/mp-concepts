# 홈 화면 상단(헤더) 이동 시 스크롤 최적화 보고서

**날짜**: 2026-03-29
**Task**: fix_home_scroll_to_top_on_header
**담당**: Antigravity

## 1. 개요
홈 화면의 하단 컨텐츠(추천, 뉴스 등)를 탐색하다가 다시 상단 프로필 아이콘으로 이동할 때, 스크롤 위치가 하단에 고정되어 있으면 최상단의 로고와 헤더 정보가 완전히 보이지 않을 수 있습니다. 이를 부드럽게 최상단으로 자동 스크롤되도록 개선하였습니다.

## 2. 주요 수정 내용

### 2.1 홈 화면 스크롤 컨테이너 제어 (`SamsungTVHome.svelte`)
- **수정**: `afterUpdate` 생명주기 함수 내부에 포커스 섹션이 'header'일 경우, `.scroll-container`의 스크롤 위치를 즉시 `0`으로 이동시키는 로직을 추가하였습니다.
- **적용**: `scrollContainer.scrollTo({ top: 0, behavior: 'smooth' });`

## 3. 테스트 및 검증 결과
- **동작**: '지금 시청하기' 버튼에서 위 방향키를 눌러 프로필 아이콘으로 포커스를 옮기면, 스크롤이 자동으로 최상단으로 올라가서 로고와 헤더 전체가 명확히 보임을 확인하였습니다.
- **예외 처리**: `header` 이외의 섹션들에 대해서는 기존과 동일하게 해당 섹션이 중앙에 오도록 스크롤링되는 로직을 유지하였습니다.

## 4. 최종 결과물 소감
사용자가 홈 화면의 최하단에서도 한 번에 최상단으로 포커스를 올렸을 때, 시각적으로 즉시 전체 레이아웃이 원점으로 복구되는 느낌을 주어 탐색의 안정감을 높였습니다.
