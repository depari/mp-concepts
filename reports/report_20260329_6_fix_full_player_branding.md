# 앱 전환 모드(전체 화면) 플레이어 UI 브랜딩 및 검은 화면 해결 보고서

**날짜**: 2026-03-29
**Task**: fix_full_player_branding
**담당**: Antigravity

## 1. 개요
PIG 모드에서 전체 화면(앱 실행)으로 전환할 때 여전히 검은 화면이 보이던 현상을 해결하고, 각 서비스(Netflix, Disney+, YouTube 등)의 브랜드 색상과 로고가 반영된 전용 플레이어 UI를 구축하였습니다.

## 2. 주요 수정 내용

### 2.1 브랜드 특화 플레이어 UI (`SamsungTVHome.svelte`)
- **수정 전**: 단순 배경색과 텍스트만 노출되던 방식.
- **수정 후**: **모킹 플레이어(Mock Player)** 레이아웃 도입.
  - `provider_id`에 기반한 전용 브랜드 컬러 테두리(`border`)와 브랜드 로고 타이틀 표시.
  - 중앙에 프로필 연동 재생 아이콘과 타이틀, 서브타이틀(예: 에피소드 정보) 노출.
  - 하단에 재생 바(Progress Bar), 버퍼 바(Buffer Bar), 재생 시간(14:02 / 45:00) 정보 상시 노출.
  - **애니메이션**: 중앙 재생 기호에 부드러운 펄스(`pulse`) 호흡 애니메이션 적용.

### 2.2 종료 버튼 및 조작 힌트
- 우측 상단에 강렬한 색상의 `✕ STOP` 버튼과, 하단에 리모컨 조작 힌트(Rewind, Pause, Forward)를 추가하여 조작성을 강화하였습니다.

## 3. 테스트 및 검증 결과
- **동작**: PIG 화면에서 Enter를 누르면 순식간에 확장되며 해당 브랜드의 색상이 입혀진 세련된 플레이어 화면으로 전환되는 것을 확인하였습니다.
- **가시성**: 검은 화면 현상을 완전히 제거하였으며, ESC(Back) 키와 "STOP" 버튼이 모두 홈 화면으로 정상 작동함을 확인하였습니다.

## 4. 최종 결과물 소감
단순한 화면 확장을 넘어, 입체적인 플레이어 UI를 제공함으로써 사용자가 실제 스트리밍 서비스를 이용하고 있다는 몰입감을 선사할 수 있게 되었습니다.
