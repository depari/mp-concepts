# Android TV 카드 디자인 복구 및 내부 레이아웃 최적화 보고서

## 1. 개요
프로필 카드의 프리미엄한 외형과 배경 이미지의 임팩트를 유지하기 위해 카드 자체의 크기를 원복하고, 대신 내부 컨텐츠 배치를 조정하여 화면 상하단 시스템 문구와의 겹침을 해결하였습니다.

## 2. 주요 수정 사항
### 2.1 프로필 카드 외형 및 배경 복구
- **너비 복구**: 기본 너비 `120.dp`, 확장 너비 `450.dp`로 다시 원복하여 웹 프로토타입의 아코디언 비율을 회복했습니다.
- **배경 유지**: `fillMaxHeight`를 다시 적용하고 외곽 패딩을 제거하여, 캐릭터별 고유 그라데이션과 발광 효과가 화면 전체 높이를 채우도록 복구하였습니다.

### 2.2 카드 내부 컨텐츠(Avatar, Name) 위치 최적화
- **내부 레이아웃 이동**: 카드 자체는 화면 전체를 채우지만, 내부의 아바타와 닉네임이 포함된 `Column`에 **`top = 160.dp`, `bottom = 80.dp`**의 내부 여백을 적용하였습니다.
- **효과**: 카드의 배경 이미지는 상하단 문구 뒤에 깔리면서도, 실제 아바타와 이름 등의 정보는 타이틀/가이드 문구를 피해 중앙 하단 영역에 조화롭게 배치됩니다.

### 2.3 시스템 문구의 극단적 엣지 배치 (Slim & Edge)
- **상단 타이틀**: 상단 여백을 **`25.dp`**로 극소화하고, `headlineSmall` 폰트와 슬림한 두께(`Light`)를 적용하여 시각적 무게감을 줄였습니다.
- **하단 가이드**: 하단 여백을 **`20.dp`**로 조정하여 화면 하단 끝에 바짝 밀착시켰습니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 프로필 카드의 화려한 배경이 상단 타이틀 뒤까지 시원하게 펼쳐지는지, 그리고 아바타와 이름이 타이틀과 겹치지 않고 적절히 아래로 내려와 배치되었는지 확인하십시오.
