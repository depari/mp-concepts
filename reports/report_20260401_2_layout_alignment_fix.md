# Android TV 레이아웃 교정 및 중앙 정렬 최종 보고서

## 1. 개요
프로필 가로 목록이 우측으로 치우치는 현상과 상단 타이틀의 위치 오류를 해결하기 위해 레이아웃 구조를 전면 개편하였습니다.

## 2. 주요 수정 사항
### 2.1 수동 Offset 기반 정밀 중앙 정렬 (Horizontal Alignment)
- **TvLazyRow 한계 극복**: 아이템 개수가 적을 때 발생하는 Pivot Offset의 불확실성을 제거하기 위해 위젯 레이아웃을 `Row` + `offset` 방식으로 변경하였습니다.
- **동적 오프셋 계산**: `LocalConfiguration`을 통해 화면 너비를 획득하고, 포커스된 아이템의 인덱스와 확장 너비(450dp)를 고려하여 **아이템의 정중앙이 화면의 정중앙에 수치적으로 일치**하도록 계산식을 적용하였습니다.
  - `targetOffset = (screenWidth/2) - (expandedWidth/2) - (누적 간격)`
- **유연한 이동**: 인덱스 변화에 따라 `animateDpAsState`가 부드럽게 목록을 밀어주어 완벽한 중앙 집중형 조작감을 제공합니다.

### 2.2 상단 타이틀 레이아웃 교정 (Title Fix)
- **영역 분리**: `Box` 내부의 단순 중첩 대신 `Column` 구조와 명시적 `padding(top = 120.dp)`을 사용하여 타이틀이 화면 상단에 정갈하게 위치하도록 하였습니다. 다른 UI 요소와의 겹침 현상을 원천 차단하였습니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 앱 시작 시 및 프로필 이동 시 선택된 패널이 화면 정중앙에 정확히 위치하며, 상단 문구가 의도된 위치에 안정적으로 표시됩니다.
