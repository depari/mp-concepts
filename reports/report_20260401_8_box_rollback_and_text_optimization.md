# Android TV 레이아웃 Box 구조 롤백 및 문구 배치 최적화 보고서

## 1. 개요
컴포저블 간의 상호작용 및 시각적 일관성을 위해 수직 스택(Column) 구조를 철회하고, 다시 **Box 기반의 중첩 레이어 구조**로 롤백하였습니다. 대신 문구의 절대 위치를 재조정하여 프로필 카드와의 겹침 문제를 해결하였습니다.

## 2. 주요 수정 사항
### 2.1 Box 레이아웃 구조 복원
- 사용자의 요청에 따라 `Column` 기반의 분리 구조에서 `Box` 기반의 통합 구조로 원복하였습니다. 이를 통해 배경 효과나 카드 애니메이션이 전체 화면 좌표계를 기준으로 유연하게 동작할 수 있게 되었습니다.

### 2.2 문구 위치(Text Position) 최적화
- **상단 타이틀**: `Alignment.TopCenter`와 `padding(top = 80.dp)`를 적용하여, 프로필 카드가 확장되어도 가독성이 침해되지 않는 안전 영역으로 이동시켰습니다.
- **하단 가이드**: `Alignment.BottomCenter`와 `padding(bottom = 50.dp)`를 적용하여 화면 하단에 안정적으로 배치하였습니다.

### 2.3 정밀 중앙 정렬 로직 보존
- 롤백 과정에서도 `Manual Offset Calculation` 로직은 그대로 유지하여, 아코디언 너비가 변하더라도 포커스된 카드가 화면 정중앙에 위치하는 기능은 완벽하게 작동합니다.

## 3. 결과 확인
- 빌드 결과: **BUILD SUCCESSFUL**
- 확인 사항: 화면 레이아웃이 `Box` 기반으로 유연하게 돌아왔는지, 상하단 문구의 위치가 카드와 겹치지 않고 적절한 공간을 확보했는지 확인하십시오.
