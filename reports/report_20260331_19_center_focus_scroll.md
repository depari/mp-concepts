# Android TV 프로필 중앙 정렬 및 디폴트 포커스 구현 보고서

## 1. 개요
웹 프로젝트 상의 '포커스 항목 중앙 정렬' 및 '앱 시작 시 자동 포커스' 기능을 안드로이드 TV 환경에 최적화하여 구현하였습니다.

## 2. 주요 구현 내용
### 2.1 중앙 정렬 스크롤 (Center Alignment)
- **TvLazyRow 도입**: 안드로이드 TV 전용 지연 로딩 행(`TvLazyRow`)을 사용하여 수천 개의 아이템이 있어도 부드러운 성능을 보장하도록 하였습니다.
- **Pivot Offset 설정**: `pivotOffsets = PivotOffsets(0.5f)` 를 적용하여 사용자가 좌우로 이동할 때마다 **포커스된 프로필 패널이 항상 화면의 중앙**에 오도록 동적인 스크롤 위치를 고정하였습니다.

### 2.2 지능형 디폴트 포커스 (Default Focus)
- **FocusRequester 연동**: 화면이 로드되는 즉시 `LaunchedEffect`를 통해 첫 번째 프로필(지은)에 포커스를 선점하도록 로직을 추가하였습니다.
- **사용자 경험(UX)**: 앱 실행 시 별도의 조작 없이도 첫 번째 프로필이 즉시 확장(Accordion)된 상태로 노출되어 프리미엄한 첫인상을 제공합니다.

### 2.3 코드 구조 개선
- **MainActivity.kt**: `TvLazyRow` 상태 관리 및 포커스 요청 로직 통합.
- **ProfilePanel.kt**: 외부로부터 `modifier`를 주입받아 포커스 타겟을 명확히 정의함.
