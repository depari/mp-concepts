# Android TV POC: Phase 2 데이터 레이어 및 컴포넌트 고도화 보고서

## 1. 개요
프로필 선택 이후의 사용자 여정을 지원하기 위해, 데이터 모델을 정립하고 홈 화면의 핵심 UI 컴포넌트인 추천 카드(RecommendedCard)를 웹 프로토타입 수준으로 구현하였습니다.

## 2. 주요 작업 내용
### 2.1 아키텍처 및 패키지 구조 최적화
- **ViewModel 분리**: `MainActivity`에 인라인으로 존재하던 로직들을 `com.depari.mpconcepts.viewmodel` 패키지로 분리(`ProfileViewModel`, `HomeViewModel`)하여 관심사를 분리하고 유지보수성을 높였습니다.
- **데이터 모델 동기화**: 웹의 `profiles.ts` 및 추천 시스템과 호환되는 `Profile` 및 `Content` 데이터 타입을 정의하였습니다. (장르, 진행률, 테마 컬러 등 포함)

### 2.2 프리미엄 컴포넌트: `RecommendedCard`
- **시네마틱 디자인**: 이미지 배경, 하단 텍스트 그라데이션 오버레이, 그리고 시청 중인 콘텐츠를 위한 진행률 바(ProgressBar)를 구현하였습니다.
- **TV 최적화 애니메이션**: 포커스 시 1.1배 확대 및 테두리 발광 효과를 통해 리모컨 조작 시의 시인성을 극대화하였습니다.

### 2.3 시스템 안정성 확보 (TDD)
- 누적 오차에 의한 프로필 리스트 치우침 현상을 픽셀 기반 연산으로 해결하였습니다.
- 패키지 이동 및 데이터 모델 변경에 따른 모든 단위 테스트(Unit Tests)를 보정하여 빌드 무결성을 확보하였습니다.

## 3. 향후 계획
- **홈 화면 레이아웃 고도화**: `LazyRow`를 중첩하여 웹의 다양한 추천 섹션(추천, 최신, 장르별)을 완성합니다.
- **전환 애니메이션**: 프로필 선택 시 홈 화면으로 진입하는 과정에 시네마틱 전환 효과를 추가합니다.
