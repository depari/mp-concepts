## 1. 홈 화면 컨텐츠 섹션 풀(Full) 연동 및 설계 개선
* **문제 상황**: 홈 화면(`SamsungTVHome`)에 최근 시청, 추천 영화, 뉴스 등의 섹션이 정상적으로 노출되지 않고 있었음. 이전 데이터 이원화 작업 과정에서 스토어는 분리했으나 UI 컴포넌트 연동이 누락됨.
* **개선 내용**:
  - **홈 전용 스토어 보강**: `homeRecommendedContents` 스토어를 신규 추가하고, `Art Store` 등 공통 추천 컨텐츠를 도입함.
  - **UI 컴포넌트 결합**: `SamsungTVHome.svelte` 에 `RecentWatchRow`, `RecommendedRow`, `NewsCardRow`를 배치함. 
  - **스크롤 안정화**: 홈 화면 컨텐츠가 많아짐에 따라 `scroll-container`를 도입하고 스타일을 정비하여, 메인 히어로 영역과 하단 행들이 자연스럽게 스크롤되도록 함.
  - **테스트 무결성 확보**: 중복 텍스트(예: Hero와 카드에 모두 표시되는 삼성 가이드 제목)에 대응하도록 테스트 쿼리를 `getAllByText` 등으로 개선하여 100% Pass 달성.
* **최종 UX**: 
  - 사용자는 홈 화면 첫 진입 시 시스템 공통 앱(Settings 등)과 자신의 전용 컨텐츠를 한곳에서 편리하게 확인 가능.
  - 탐색창과 홈 화면의 풍성함이 동일 수준으로 맞춰져 일관성 있는 UX 완성.

결과적으로 개인화된 스마트 TV의 ‘홈’으로서 기능을 완성하였습니다.
