## 1. 앱 전체 화면(Full Screen) 진입 시 검은 화면 및 종료 편의성 개선
* **문제점**:
  - PIG 모드에서 전체 화면으로 전환 시, 배경 그라데이션 정보가 없는 컨텐츠(예: 일반 앱 등)의 경우 검은 화면만 노출되어 사용자가 앱이 멈춘 것으로 오해할 수 있음.
  - 전체 화면 상태에서 홈으로 돌아가기 위해 리모컨의 ESC(Back) 키 외에는 시각적인 종료 방법이 제공되지 않음.
* **개선 계획**:
  - `SamsungTVHome.svelte`의 전체 화면 UI(`app_running`)에서 `pigContent`의 `thumbnail_gradient`가 없을 경우 `iconColor`를 배경으로 사용하거나 기본 어두운 배경색을 적용하도록 폴백 로직 추가.
  - 우측 상단에 시각적인 "✕ 닫기 (홈으로)" 버튼을 추가하여 마우스/포인터 조작 시에도 종료 가능하도록 개선.
* **최종 결과**:
  - 앱 컨텐츠(Netflix 등) 실행 시 해당 서비스의 브랜드 색상이 배경으로 노출되어 검은 화면 문제가 해결됨.
  - "닫기" 버튼을 통해 직관적으로 홈 화면으로 복귀 가능함.
