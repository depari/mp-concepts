## 1. 홈 화면 신규 섹션(최근 시청 등) 포커스 이동 및 조작 개선
* **문제 상황**: 홈 화면에 최근 시청, 추천 영화 등의 섹션을 추가했으나, 리모컨 방향키 조작 시 해당 섹션으로 포커스가 이동하지 않고 기존 섹션(Header, Hero, Apps) 사이에서만 맴도는 현상 발생.
* **개선 내용**:
  - **네비게이션 맵 확장**: `homeNavigationStore.ts`의 `SECTIONS` 리스트에 `recents`, `recommended`, `news`를 추가하고 상하/좌우 이동 로직을 전체 6개 섹션에 맞게 재설계함.
  - **데이터 개수 실시간 연동**: `navigation.ts`에서 `moveHomeFocus` 호출 시 각 섹션의 아이템 개수를 `get()` 스토어를 통해 실시간으로 전달하여, 섹션 내 카드 이동 범위를 정확히 제한함.
  - **비주얼 강조 및 자동 스크롤**: 
    - 포커스된 섹션 타이틀이 강조(`active-section`)되도록 CSS를 보강함.
    - `afterUpdate` 라이프사이클을 이용해 포커스된 섹션이 항상 화면 중앙에 오도록 `scrollIntoView` 자동 스크롤 로직을 구현함.
* **최종 UX**: 
  - 사용자는 홈 화면에서 화살표 키만으로 새로 추가된 시청 기록과 추천 뉴스 사이를 자유롭게 오갈 수 있음.
  - 섹션 이동 시 화면이 부드럽게 스크롤되어 현재 조작 중인 위치를 놓치지 않음.

결과적으로 풍성해진 홈 화면의 모든 요소를 리모컨으로 조작 가능한 상태로 완성하였습니다.
