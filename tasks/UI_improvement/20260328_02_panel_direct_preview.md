# 포커스 상태 시 패널 직접 프리뷰 및 하단 탐색 힌트 표출 개선

* **Date**: 2026-03-28
* **Task Number**: 02
* **Memorized Item**: panel_direct_preview

## 개요
기존 Full Mode(`ProfileScreen`) 환경에서는 엔터 키(Enter)를 입력해야 이어서 볼 콘텐츠(recentContents)가 나열되었습니다. 
이를 수정하여, **프로필 포커스를 받는 즉시 주요 선호 앱들과 이어보기 항목이 표시**되도록 UX를 강화했습니다.
더불어 전체화면 대시보드 진입을 직관적으로 유도하기 위해 하단 방향키 힌트 문구를 추가했습니다.

## 개선 과정
1. **상태 의존성 제거**: `ProfilePanel.svelte` 컴포넌트 내 `showContents` 상태의 토글 의존성을 삭제. 포커스가 `true`일 경우 조건 없이 하위 콘텐츠가 자동으로 노출되도록 변경.
2. **앱 마커 추가**: 포커스 시 프로필 아바타 아래에 간단히 선호하는 주요 앱(Netflix, YouTube, Prime, Tving 등) 브랜드 컬러 아이콘 라인업을 `{#each}` 블록으로 추가 디자인하여 시각적 정보를 증대시킴.
3. **가이드 힌트 개선**: 기존 단순했던 텍스트("Enter — 이어보기")를 제거하고 `↓ 상세 항목 확인` 문구와 함께 화살표의 상하 바운싱(CSS Keyframes bounce) 애니메이션을 부여해 심미성과 탐색 힌트 기능을 결합시킴.
4. **잔여 코드 제거 (TC/Lint 최적화)**: 기능 축소로 인해 더 이상 사용하지 않게 된 `panel__wave` 관련 CSS 및 마크업을 제거.

## 상세 결과
- 불필요하게 깊었던 인터랙션 뎁스(Enter 키 의존도)를 제거하고 직관적인 프리뷰 경험으로 고도화했습니다.
- `Vitest` 테스트 상호 토글 및 포커스 관리 로직 상에도 사이드 이펙트 없이 안정적으로 구동됩니다.
