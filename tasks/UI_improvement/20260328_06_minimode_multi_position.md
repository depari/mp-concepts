# 미니 모드 4-way 레이아웃 포지셔닝 기능 추가

* **Date**: 2026-03-28
* **Task Number**: 06
* **Memorized Item**: minimode_multi_position

## 개요
기존에 화면 하단에서만 활성화되던 미니 모드(프로필 빠른 전환 화면)를, 사용자의 편의성에 맞게 **상단(Top), 하단(Bottom), 좌측(Left), 우측(Right)** 총 네 가지 방향에 자유자재로 띄울 수 있도록 위치 전환 기능을 개발 완료했습니다.

## 구현 상세 내용
1. **스토어 상태 추가 (`miniModeStore.js`)**:
   - `isActive` 프로퍼티 위주에서, 레이아웃 메타데이터를 담은 `position: 'bottom'` 상태 변수 및 `setMiniModePosition(pos)` 함수를 등록했습니다.
2. **동적 레이아웃 CSS 반영 (`MiniModeSelector.svelte`)**:
   - 위치(`pos`)에 따라 플렉스 축(Flex-direction)이 수평(가로 바)/수직(세로 바)으로 동적 변환되도록 다중 CSS 클래스를 구현했습니다.
   - 포커스 시 강조되는 아바타의 방향 또한 하단일 때는 위로 뛰고(`translateY`), 좌측일 때는 우로 뛰는(`translateX`) 식별 로직을 동적으로 연산하도록 CSS 마진을 쪼갰습니다.
3. **위치 제어 버튼(UI) 삽입**:
   - 미니 모드 헤더 우측('✕' 닫기 버튼 옆)에 [T], [L], [R], [B] 4개의 직관적인 위치 제어 스위치 버튼을 추가하여, 실시간으로 패널 위치를 날아다니듯(Fly Transition) 변경할 수 있습니다.

## 상세 결과
해당 기능 도입으로 TV 스크린 내 뒷배경 콘텐츠를 가리지 않는 방향을 사용자가 능동적/직접적으로 선택할 수 있어 프로필 퀵 체인지 UX가 크게 증대되었습니다.
