# 작업 보고서 [2026-03-28]
## 작업 내용: 미니 모드 측면 배치 시 탐색 방향키 개선

### 1. 개요
미니 모드가 좌측 또는 우측 사이드에 배치되었을 때, 시각적인 구조에 맞게 상하 방향키(`ArrowUp`, `ArrowDown`)를 사용하여 프로필을 탐색할 수 있도록 개선 요청되었습니다.

### 2. 변경 사항
- **`src/lib/utils/navigation.js`**
    - `miniModeStore`를 참조하여 현재 미니 모드의 활성화 여부 및 위치를 확인합니다.
    - 미니 모드가 좌측(`left`) 또는 우측(`right`)에 위치한 경우:
        - `ArrowUp` 키를 누르면 이전 프로필로 이동(`navigate(-1)`)하도록 설정했습니다.
        - `ArrowDown` 키를 누르면 다음 프로필로 이동(`navigate(1)`)하도록 설정했습니다.
        - 기존의 좌우 키(`ArrowLeft`, `ArrowRight`)는 동작하지 않도록 처리하여 혼선을 방지했습니다.
    - 미니 모드가 하단/상단이거나 일반 Full 모드인 경우:
        - 기존 방식과 동일하게 좌우 키(`ArrowLeft`, `ArrowRight`)로 프로필을 탐색합니다.
        - `ArrowUp`, `ArrowDown`은 대시보드 활성화/비활성화 기능을 유지합니다.

### 3. 결과
- 미니 모드 위치에 따른 직관적인 키 내비게이션 환경을 제공하게 되었습니다.
- 시각적인 정렬 방향(세로형)에 맞는 조작감을 확보했습니다.
