# 작업 보고서 [2026-03-28]
## 작업 내용: 3D 프로필 아바타 고도화 (상반신/얼굴 위주 포트레이트)

### 1. 개요
프로필 선택 화면의 시인성을 높이기 위해, 기존 전신/상반신 위주의 3D 아바타를 얼굴과 어깨선 중심의 **클로즈업 포트레이트(Close-up Portrait)** 스타일로 고도화했습니다.

### 2. 변경 사항
- **아바타 에셋 재생성**
    - 얼굴의 디테일과 표정이 더 잘 보이도록 클로즈업 된 7종의 3D 아바타를 새롭게 생성했습니다.
    - `public/avatars/` 내 기존 파일들을 새로운 고화질 포트레이트 이미지로 교체 완료했습니다.
- **UI 최적화**
    - 아바타가 더 크게 강조됨에 따라 프로필 원형 프레임 내에서 인물이 더 뚜렷하게 보이도록 조정되었습니다.
    - 테마와 컬러 구성은 이전 작업에서 확립된 파스텔 톤을 유지하여 일관성을 확보했습니다.

### 3. 결과
- 프로필 아이콘이 작게 표시되는 '미니 모드'나 '탐색 화면'에서도 실사용자의 얼굴을 훨씬 직관적으로 식별할 수 있게 되었습니다.
